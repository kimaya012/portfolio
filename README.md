## ğŸš€ Getting Started

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/kimaya012/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your preferred browser
   - Or use a local server like Live Server in VS Code

3. **Make modifications**
   - Edit content in `index.html`
   - Customize styles in `assets/css/style.css`
   - Add functionality in `assets/js/script.js`

## ğŸ“ˆ GitHub Pages Deployment

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial portfolio commit"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to Pages section
   - Select source: Deploy from a branch
   - Choose branch: main / (root)
   - Save settings

3. **Access your site**
   - Your portfolio will be available at: `https://kimaya012.github.io/portfolio`
